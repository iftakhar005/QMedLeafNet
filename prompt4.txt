            7. Pass through backbone: frozen MobileNetV3-Small

SECTION 5: DATA TRANSFORMATIONS AT EACH STEP
================================================================================

5.1 IMAGE DATA TRANSFORMATIONS

Original Image (disk file)
  ↓
  Input: "data/raw/Augmented Dataset/Aloe Vera Disease/img_001.jpg"
  Format: JPG, 1440×1080 pixels, RGB
  Size: ~150 KB
  ↓
  Step 1: Load Image (PIL.Image.open)
  Output: PIL Image object, RGB, (1440, 1080, 3)
  ↓
  Step 2: Resize with Padding
  Calculate scale: min(224/1440, 224/1080) = min(0.156, 0.207) = 0.156
  Resized to: (225, 169) pixels
  Center-pad to: (224, 224) with black border
  Output: PIL Image, RGB, (224, 224, 3)
  ↓
  Step 3: Convert to Numpy Array (HWC format)
  Output: numpy array, shape (224, 224, 3), dtype uint8, values [0-255]
  ↓
  Step 4: Transpose to CHW
  Output: numpy array, shape (3, 224, 224), dtype uint8
  ↓
  Step 5: Normalize to [0, 1]
  Output: numpy array, shape (3, 224, 224), dtype float32, values [0.0-1.0]
  ↓
  Step 6: ImageNet Normalization
  For each channel C:
    normalized_C = (channel_C - mean_C) / std_C
  Means: [0.485, 0.456, 0.406]
  Stds: [0.229, 0.224, 0.225]
  Output: numpy array, shape (3, 224, 224), dtype float32, values [-2.5 to 2.5]
  ↓
  Step 7: Convert to PyTorch Tensor
  Output: torch.FloatTensor, shape (1, 3, 224, 224) after batch dimension
  ↓
  Step 8: Pass Through Frozen Backbone
  MobileNetV3-Small Feature Extractor
  Input: (1, 3, 224, 224)
  Processing: Depthwise-separable convolutions + pooling (927K frozen params)
  Global Average Pooling: reduces spatial dimensions
  Output: (1, 576) feature vector
  ↓
  Step 9: Extract Feature Vector
  Output: numpy array, shape (576,), dtype float32, values [-2.5 to 2.5]
  ↓
  Final: Batch 1,600 features together
  Output: numpy array, shape (1600, 576), dtype float32

5.2 LABEL DATA TRANSFORMATIONS

Input: Species + Condition names
  Example: species="Aloe Vera", condition="Disease"
  ↓
  Step 1: Map species name to index [0-6]
  Species list: [Aloe Vera, Azadirachta, Centella, Hibiscus, Kalanchoe, Mikania, Piper]
  "Aloe Vera" → species_idx = 0
  ↓
  Step 2: Map condition name to index [0-8]
  Condition list: [Healthy, Disease, Chlorotic, Insects, Mild Disease, Dried, Young Healthy, Mature Healthy, Distorted]
  "Disease" → condition_idx = 1
  ↓
  Step 3: Compute combined label
  label = condition_idx * 7 + species_idx
  label = 1 * 7 + 0 = 7
  ↓
  Final: Batch 1,600 labels together
  Output: numpy array, shape (1600,), dtype int64, values [0-62]

5.3 FEATURE-LABEL PAIR TRANSFORMATION

(1600 images) + (labels) → (features, labels)
  ↓
  Extract all 1600 image features:
    Output: (1600, 576) float32 array
  ↓
  Create all 1600 labels:
    Output: (1600,) int64 array
  ↓
  Together they form training data:
    X_train: (1600, 576)
    y_train: (1600,)
  ↓
  Same for validation (400 samples):
    X_val: (400, 576)
    y_val: (400,)
  ↓
  These are fed to the 4 classical heads for training

5.4 MODEL OUTPUT TRANSFORMATIONS

Logistic Regression Head:
  Input: (400, 576) float32 features
  ↓
  StandardScaler normalize: (400, 576)
  ↓
  Linear layer: Linear(576, 63) → (400, 63) logits
  ↓
  Predictions: argmax(logits, dim=1) → (400,) labels [0-62]
  ↓
  Probabilities: softmax(logits, dim=1) → (400, 63) [0, 1]

SVM (RBF/Linear) Heads:
  Input: (400, 576) float32 features
  ↓
  StandardScaler normalize: (400, 576)
  ↓
  SVM decision function: (400,) decision values
  ↓
  Predictions: argmax(decision values) → (400,) labels [0-62]
  ↓
  Probabilities: calibrated probabilities → (400, 63) [0, 1]

Dense MLP Head:
  Input: (400, 576) float32 features as torch tensor
  ↓
  Linear(576, 64): (400, 64)
  ↓
  ReLU activation: (400, 64)
  ↓
  Dropout(0.1): (400, 64)
  ↓
  Linear(64, 63): (400, 63) logits
  ↓
  Predictions: argmax(logits, dim=1) → (400,) labels [0-62]
  ↓
  Probabilities: softmax(logits, dim=1) → (400, 63) [0, 1]

5.5 METRICS TRANSFORMATIONS

Predictions (400,) + Ground Truth (400,) → Metrics
  ↓
  Accuracy: sum(pred == truth) / 400 → scalar [0.0-1.0]
    Example: 195 correct / 400 = 0.4875 (48.75%)
  ↓
  F1-Score (weighted): sklearn.metrics.f1_score(truth, pred, average='weighted')
    Accounts for class imbalance
    Output: scalar [0.0-1.0]
  ↓
  Precision (weighted): sklearn.metrics.precision_score(truth, pred, average='weighted')
    TP / (TP + FP) averaged across classes
    Output: scalar [0.0-1.0]
  ↓
  Recall (weighted): sklearn.metrics.recall_score(truth, pred, average='weighted')
    TP / (TP + FN) averaged across classes
    Output: scalar [0.0-1.0]
  ↓
  Create metrics dictionary: {
    'accuracy': 0.4875,
    'f1_score': 0.487,
    'precision': 0.491,
    'recall': 0.487
  }
  ↓
  Repeat for all 4 heads:
    Head 1 metrics + Head 2 metrics + Head 3 metrics + Head 4 metrics
  ↓
  Output: Aggregated dictionary with all 4 heads' metrics

SECTION 6: PREPROCESSING PIPELINE DETAILED BREAKDOWN
================================================================================

PREPROCESSING STEP 1: LOAD IMAGE RGB
  Input: File path string
  Process:
    image = PIL.Image.open(file_path)
    image = image.convert('RGB')
  Output: PIL Image object with 3 RGB channels
  Error handling: If file doesn't exist, return None
  Time: ~10 ms per image

PREPROCESSING STEP 2: RESIZE WITH PADDING
  Input: PIL RGB image (variable size, usually 1440×1080)
  Process:
    target_size = (224, 224)
    original_width, original_height = image.size
    scale = min(224 / original_width, 224 / original_height)
    new_width = int(round(original_width * scale))
    new_height = int(round(original_height * scale))
    resized_image = image.resize((new_width, new_height), BILINEAR)
    canvas = new blank image (224, 224) filled with black (0, 0, 0)
    offset_x = (224 - new_width) // 2
    offset_y = (224 - new_height) // 2
