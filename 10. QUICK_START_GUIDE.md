## 📞 Common Issues & Quick Fixes

### Python not found
```
python3 --version
# If not found, install Python 3.9+ from python.org
```

### Package installation fails
```
# Try upgrading pip first
pip install --upgrade pip
pip install -r requirements_phase1.txt
```

### Disk space too small
```
# Check what you need
du -sh data/raw/  # See raw data size
# Free up space or use external drive
```

### Mendeley download fails
```
# Try:
1. Clear browser cache
2. Use different browser
3. Check internet connection
4. Contact Mendeley support
```

### Filename patterns don't match
```
# Show Claude actual samples:
find data/raw -name "*.jpg" -type f | head -20
# Claude will adjust extraction regex
```
